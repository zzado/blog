**Astro**는 콘텐츠 중심 웹사이트를 위한 최고의 웹 프레임워크입니다.

## 주요 특징

1.  **Zero JS by Default:** 불필요한 자바스크립트를 제거하여 로딩 속도가 매우 빠릅니다.
2.  **Islands Architecture:** 필요한 부분만 동적으로 작동하게 만들 수 있습니다.
3.  **UI Agnostic:** React, Vue, Svelte 등 좋아하는 프레임워크를 그대로 사용할 수 있습니다.

## 설치 방법

```bash
npm create astro@latest
```

이제 여러분도 시작해보세요!
