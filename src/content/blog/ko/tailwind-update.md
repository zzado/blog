**Tailwind CSS 4.0**이 드디어 공개되었습니다! 🎉

## 무엇이 달라졌나요?

*   **성능 향상:** 기존 버전 대비 빌드 속도가 획기적으로 빨라졌습니다.
*   **간소화된 설정:** `tailwind.config.js` 없이도 CSS 파일 하나로 설정이 가능합니다.
*   **새로운 엔진:** Rust 기반의 Oxide 엔진을 탑재했습니다.

이제 CSS 작성이 더 즐거워질 것입니다.
